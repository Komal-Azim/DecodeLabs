# DecodeLabs
Project week 2
Project 2: Basic Classification Model using Iris Dataset

This project demonstrates a simple supervised machine learning classification model using the famous Machine Learning concept with the Iris Dataset.
The goal of this project is to understand the complete workflow of a basic classification task, including data loading, preprocessing, train-test splitting, model training, and evaluation.

Features
Load and explore the Iris dataset
Understand dataset structure and features
Split data into training and testing sets
Train a basic classification algorithm
Predict flower species based on input features
Evaluate model performance using accuracy score
Technologies Used
Python
Pandas
NumPy
Scikit-learn
Matplotlib / Seaborn (optional for visualization)
Machine Learning Concepts Covered
Supervised Learning
Classification
Model Training
Train-Test Split
Prediction and Evaluation
Dataset Information

The Iris dataset contains measurements of iris flowers:

Sepal Length
Sepal Width
Petal Length
Petal Width

Target Classes:

Iris Setosa
Iris Versicolor
Iris Virginica
